Como Usarlo

Touch and Go.js es un plugin muy sencillo que te permite desplegar una caja oculta, con algún contenido, hacia un costado para poder visualizarlo con solo hacerle un click al botón que asignes. Para poder utilizarlo tienes que seguir los siguientes pasos:

En primer lugar, descarga todo el contenido del pluglin para luego , mediante la etiqueta <script>, ubicar los archivos en el HTML como en el ejemplo de la derecha:

A continuación, necesitas tener una estructura HTML como en el ejemplo de la derecha. Cabe decir que se necesitan tener dos elementos principales; primero una etiqueta con la class “expandir1” y luego una caja contenedora con una class como “caja-abierta1” (si se utiliza varias veces, es recomendable utilizar números para que el efecto funcione con la caja especifica)

Finalmente, si observamosel archivo .js, básicamente dice que una ves cargado el documento, al hacer click en la etiqueta con la class “expandir1”, se expanda y aparezca el contenedor con la class “caja-abierta1”. Una ves abierta, si retiras el mouse de dicho contenedor, este se oculta.

*NOTA:
El nombre de las class puede variar dependiendo de tus necesidades, puedes copiar y pegar el codigo JQuery para conectar con varios elementos y contenedores.
También, es recomendable posicionar la etiqueta y la caja contenedora con las clases, una arriba de la otra en el HTML.
ES NECESARIO que de antemano se le asigne un "display: none;" al contenedor a mostrar
